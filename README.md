## 🎯 Project Overview

**OnePiece Dezato** is a premium, modern, and fully-featured e-commerce cafe website for premium waffles, pancakes, desserts, and coffee & beverages. Built with cutting-edge technologies, it provides an exceptional customer experience with admin management capabilities.

## 🛠️ Tech Stack

- **Frontend**: React 18, Next.js 14, TypeScript 5, Tailwind CSS 3
- **Backend**: Firebase (Auth, Firestore, Storage, Hosting)
- **State**: Zustand
- **Forms**: React Hook Form + Zod
- **UI**: Custom components + Lucide React Icons

## 🚀 Quick Start

### Prerequisites
- Node.js >= 18.0.0
- npm >= 9.0.0
- Firebase project

### Installation

```bash
# Clone repository
git clone https://github.com/onepiece0609/onepiece-dezato.git
cd onepiece-dezato

# Install dependencies
npm install

# Setup environment variables
cp .env.example .env.local
# Edit .env.local with your credentials

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)
